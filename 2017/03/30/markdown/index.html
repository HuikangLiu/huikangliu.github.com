#Markdown常用语法及基本工具

下面是常用的，不够再补充

  • ＃号后跟一号标题 ， ＃＃后二号标题，一次类推，字号递减*
    >
    #1
    ##2
  • ＊号后为无序列表， 1. 后为有序列表*
    >
    1.第一个
  • ‘>’后默认为斜体

    我是引用我是斜体

  • 超链接为［显示文本］(具体链接)

    markdown

  • 图片为 ![名字]（具体链接）

  • ＊＊包含为加粗，＊为斜体

    我是加粗的 我是加斜的

  • 列表长得比较麻烦，忘了看md文件里咋写的，第一个分割线不含：，中间的左右均含，最后一个末尾含
    >
    |School |Major |My GPA |
    |———-|:——————-:|——:|
    |UCL |Information Security |4.0 |

  • 代码框是用｀括起来

    print 'hello world'

  • 分割线是使用三个＊号
